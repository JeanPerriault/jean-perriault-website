# Jean Perriault Website

Here are the build step to run this website locally.

## Build Setup

``` bash
# Install dependencies
$ npm install

# Serve with hot reload at localhost:3000
$ npm run dev

# Build for production and launch server
$ npm run build
$ npm run start

# Generate static project
$ npm run generate
```
